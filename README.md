# YabonLaPub Admin Platform – Internship Project

## Overview

This repository documents the work completed during my Bachelor's Degree Internship in Computer Science at the University of Lorraine.

The internship was carried out at **YabonLaPub**, an association whose objective is to help associations raise funds through sponsored advertising campaigns.

Due to confidentiality and intellectual property restrictions, the source code developed during the internship cannot be publicly shared. This repository serves as a portfolio project containing the internship report, presentation materials, project documentation, and a summary of the work accomplished.

---

## About YabonLaPub

YabonLaPub is a platform designed to connect:

* Associations seeking financial support
* Sponsors and advertisers looking for visibility
* Users willing to support associations through their interactions with advertisements

The platform allows users to choose associations they want to support and interact with sponsored advertisements (videos and images). These interactions generate revenue that is later distributed to the selected associations.

### Back-End Integration

After receiving access to an existing back-end developed by another intern, I:

* Fixed routing issues
* Corrected database inconsistencies
* Improved controllers
* Added missing fields required by business needs
* Updated API endpoints
* Connected all front-end forms to the back-end

The final result was a fully functional and integrated administration platform.

---

### Security Improvements

I worked on authentication mechanisms using JWT tokens.

One issue discovered during testing was that expired tokens left administrators connected without access to protected actions.

To improve the user experience, I implemented:

* Token expiration detection
* Automatic logout
* Redirection to the login page
* User notification messages

---

### Advertising Statistics Module

I contributed to the implementation of a statistics dashboard allowing administrators to monitor:

* Advertisement views
* Advertisement clicks
* Campaign performance indicators

The front-end interface was completed and tested using mock data while preparing for integration with Redis-based statistics services.
